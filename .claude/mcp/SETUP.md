# MCP Setup — API Keys

Thêm vào ~/.zshrc (không commit vào git):

```bash
# GitHub MCP
export GITHUB_PERSONAL_ACCESS_TOKEN="ghp_..."

# Sentry MCP
export SENTRY_TOKEN="sntrys_..."

# Figma MCP
export FIGMA_TOKEN="figd_..."
```

Sau khi thêm: source ~/.zshrc

Kiểm tra MCP đang chạy: /mcp trong Claude Code session
