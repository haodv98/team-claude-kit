# Memory & Sessions

## Session file format
Lưu tại .claude/sessions/YYYY-MM-DD-HHMM.md

## Bắt đầu session mới
ccstart
/using-superpowers
"Đọc .claude/sessions/[file mới nhất].md và tiếp tục"

## Project memory (.claude/memory.md)
Khác session memory — tích lũy theo tháng.
Chứa: gotchas, decisions, workarounds, patterns.

Sau khi fix bug khó:
"Lưu lesson vào .claude/memory.md: [mô tả]"

Sau sprint:
"Cập nhật .claude/memory.md với decisions trong sprint này"

## Nguyên tắc
"Quên conversation, nhớ lessons"
Session memory = ngắn hạn (2 ngày)
Project memory = dài hạn (tích lũy mãi)
